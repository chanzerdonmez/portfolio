Winéko

Winéko est un site e-commerce proposant une large gamme de produits à prix réduit. Le site intègre diverses fonctionnalités essentielles telles qu'un panier d'achat, une base de données pour la gestion des produits et des utilisateurs, une plateforme de paiement sécurisée, ainsi qu'un tableau de bord pour l'administration.

Technologies utilisées : Spring Boot, Angular, MySQL
